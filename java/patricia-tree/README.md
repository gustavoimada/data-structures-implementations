Java Patricia/trie-style tree implementation for storing and displaying words.

## Highlights

- Word insertion.
- Character-based branching.
- Level-order traversal using a custom queue.
- Iterative word display using a custom stack.
- Demonstration with a classic word set such as `bear`, `bell`, `bid`, `bull`, `buy`, `sell`, `stock` and `stop`.

## Files

- `Arvore.java`: Patricia tree logic.
- `No.java`: node representation.
- `Fila.java` and `NoFila.java`: custom queue for level traversal.
- `Pilha.java` and `NoPilha.java`: custom stack for iterative display.
- `Main.java`: demo scenario.
