This folder contains Java implementations of tree-based data structures studied in Data Structures coursework.

## Projects

| Project | Description |
|---|---|
| [B+ Tree](b-plus-tree) | B+ tree implementation with insertion, deletion, split handling, underflow correction and linked leaf traversal. |
| [Binary Digital Tree](binary-digital-tree) | Binary digital tree built from fixed-size file records and binary code strings. |
| [Patricia Tree](patricia-tree) | Patricia/trie-style tree for word insertion and traversal. |
| [Patricia Trie Alternative](patricia-trie-alt) | Alternative compressed Patricia trie implementation with search and prefix handling. |
| [N-ary Tree with Linked Lists](n-ary-tree-linked-lists) | N-ary tree represented through custom linked lists for keys and child references. |
